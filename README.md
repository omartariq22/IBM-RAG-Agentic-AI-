# IBM RAG and Agentic AI Labs

A collection of hands-on labs and projects from IBM's RAG and Agentic AI Professional Certificate, exploring prompt engineering, LangChain, retrieval-augmented generation (RAG), AI agents, and LLM-powered applications.

## Local Lab Setup

The labs in this repository are being adapted so they can run locally without requiring personal IBM watsonx.ai credentials or external API keys. When a course notebook depends on watsonx.ai, the local version uses an offline study mode with deterministic sample responses so the notebook logic, prompts, LangChain templates, and exercise flow can still be practiced from VS Code.

This offline mode is meant for learning and reproducibility. It does not replace the real IBM-hosted model behavior available inside the Skills Network lab environment.

## Current Labs

- `In-Context Learning and Prompt Templates-v3-GenAIcourse.ipynb`: Prompt engineering basics, in-context learning techniques, and LangChain prompt templates with local offline execution support.
- `Build Smarter AI Apps Empower LLMs with LangChain.ipynb`: Core LangChain concepts, including models, chat messages, prompt templates, output parsers, documents, retrieval, memory, chains, tools, and agents with local offline execution support.
